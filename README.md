# SSquareG

This repository contains the full stack application for SSquareG.

## Project Structure

- `/frontend` - React application (created with Create React App)
- `/backend` - Node.js/Express server API
- `/database` - Database schemas and scripts

## Setup Instructions

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm start`

### Backend
1. `cd backend`
2. `npm install`
3. `npm run dev`

### Database
1. Run the scripts in the `database` folder to initialize the database structure.
